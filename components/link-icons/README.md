# LinkIcons

This project was created using the [Simplr Create](https://github.com/Simplr/Configs/tree/main/create) command

```bash
npm init @simplr-wc
```

## Installation

```bash
npm install link-icons
```

## Usage

```html
<script src="./src/link-icons.js" type="module"></script>

<link-icons></link-icons>
```

## Running

```bash
npm run start
```
